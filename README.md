
#  Nishi | Water & Sky

A cinematic, nature-inspired photography portfolio website built using **HTML5 and CSS3**.

This project showcases ocean, sky, and golden hour photography with a modern, minimal, and elegant design.

---

##  Live Concept

**Theme:**
Where Water Meets Sky

A visual storytelling portfolio focused on:

*  Ocean aesthetics
*  Open skies
*  Golden hour light
*  Cinematic atmosphere

---

##  Tech Stack

* HTML5
* CSS3
* Google Fonts (Playfair Display & Montserrat)
* Fully responsive design
* Flexbox & CSS Grid

---

##  Design Features

###  Hero Section

* Full-screen background image
* Dark overlay for cinematic mood
* Elegant typography
* Smooth scroll button

###  About Section

* Two-column layout (image + text)
* Soft shadow & hover scale effect
* Clean and spacious layout

###  Gallery Sections

* Ocean Series
* Sky Series
* Golden Hour Series

Features:

* CSS Grid layout
* Hover zoom animation
* Smooth spacing and alignment

###  Responsive Design

* Mobile-friendly layout
* Adaptive typography
* Stacked layout on smaller screens
